**Assistant：**

- `POST /evaluate`：接收網格狀態，隨機生成策略，執行 Policy Evaluation 計算 V(s)

前端 `templates/index.html` 與 `static/script.js` 實作了：
- n 維度選擇（5~9 下拉選單）+ Generate Grid 按鈕
- 點擊交互：第 1 次→起點（綠色），第 2 次→終點（紅色），後續→障礙物（灰色，最多 n-2 個）
- 再次點擊已設定格子可取消
- 狀態欄即時顯示設定進度

### app.py 核心邏輯（Policy Evaluation）

```python
GAMMA = 0.9
THETA = 1e-6
REWARD_GOAL = 1.0
REWARD_OBSTACLE = -1.0
REWARD_STEP = -0.04

# 隨機生成策略
for r in range(n):
    for c in range(n):
        cell = (r, c)
        if cell == end or cell in obstacles:
            policy[cell] = None
        else:
            policy[cell] = random.choice(['up','down','left','right'])

# Policy Evaluation 迭代
V = np.zeros((n, n), dtype=float)
for _ in range(10000):
    delta = 0.0
    for r in range(n):
        for c in range(n):
            if cell == end or cell in obstacles:
                continue
            action = policy[cell]
            dr, dc = ACTIONS[action]
            nr, nc = r + dr, c + dc
            # 邊界與障礙物檢查 — 撞牆留在原地
            if nr < 0 or nr >= n or nc < 0 or nc >= n:
                nr, nc = r, c
            if (nr, nc) in obstacles:
                nr, nc = r, c
            # 計算獎勵
            if (nr, nc) == end:
                reward = REWARD_GOAL
            else:
                reward = REWARD_STEP
            new_v = reward + GAMMA * V[nr, nc]
            delta = max(delta, abs(new_v - V[r, c]))
            V[r, c] = new_v
    if delta < THETA:
        break
```
